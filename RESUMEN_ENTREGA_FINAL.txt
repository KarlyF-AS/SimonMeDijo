RESUMEN EJECUTIVO
=================

Problema:
  Integrar MongoDB en SimonMeDijo manteniendo SharedPreferences y SQLite

Solucion Propuesta:
  Arquitectura multi-backend con RecordsManager central que coordina:
  - SharedPreferences (local)
  - SQLite/Room (local SQL)
  - MongoDB Realm (local NoSQL - v1.1)
  - MongoDB Atlas (remoto - v1.2)

Beneficios:
  + 100% compatible con codigo existente
  + Persistencia en cloud
  + Sincronizacion offline-first
  + Multiples usuarios
  + Auditoria centralizada

COMO USAR

  - 1 issue por agente IA

Paso 3: Ejecutar issue
  - Agente lee completamente el issue
  - Crea rama: git checkout -b feature/[nombre-fase]
  - Implementa exactamente como especifica
  - Ejecuta: ./gradlew test
  - Commit: git commit -m "feat: [titulo] (#N)"

Paso 4: Merge
  - PR a develop
  - Merge cuando PR aprobado
  - Siguiente agente toma siguiente issue
